子数组 / 子字符串的区间问题（最长无重复子串、最小覆盖子串、长度最小的子数组）

```java
/**
 * 同向双指针（滑动窗口）通用模板
 * @param nums 数组/字符串
 * @param target 目标值（和/长度/条件）
 * @return 符合条件的结果
 */
public int slidingWindow(int[] nums, int target) {
    // 1. 初始化：左指针=0，结果变量，临时变量（和/计数）
    int left = 0;
    int result = 0;
    int windowSum = 0; // 窗口内元素和（根据场景调整）
    
    // 2. 右指针遍历整个数组（扩张窗口）
    for (int right = 0; right < nums.length; right++) {
        // 3. 加入当前元素到窗口
        windowSum += nums[right];
        
        // 4. 收缩窗口：当窗口满足条件时，移动左指针
        while (windowSum >= target) { // 条件根据场景调整（> / < / 包含目标）
            // 5. 更新结果（最小窗口/最大窗口）
            result = Math.min(result, right - left + 1); // 示例：最小窗口长度
            
            // 6. 移出左指针元素，收缩窗口
            windowSum -= nums[left];
            left++;
        }
    }
    
    // 7. 返回结果（需处理无结果的情况）
    return result == 0 ? -1 : result;
}
```

经典例题：长度最小的子数组

```java
// LeetCode 209：找和≥target的最短子数组长度
public int minSubArrayLen(int target, int[] nums) {
    int left = 0;
    int sum = 0;
    int minLen = Integer.MAX_VALUE;
    
    for (int right = 0; right < nums.length; right++) {
        sum += nums[right];
        
        // 满足条件时收缩窗口，找最小值
        while (sum >= target) {
            minLen = Math.min(minLen, right - left + 1);
            sum -= nums[left];
            left++;
        }
    }
    
    return minLen == Integer.MAX_VALUE ? 0 : minLen;
}
```
