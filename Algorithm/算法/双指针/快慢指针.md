链表 / 数组的遍历问题（链表找环、删除重复元素、移除数组元素、反转链表）

```java
/**
 * 快慢指针通用模板
 * @param nums 数组/链表
 * @return 处理后的结果
 */
public int removeDuplicates(int[] nums) { // 示例：移除有序数组重复元素
    // 1. 边界处理
    if (nums == null || nums.length == 0) return 0;
    
    // 2. 初始化：慢指针（记录有效位置），快指针（遍历）
    int slow = 0;
    for (int fast = 1; fast < nums.length; fast++) {
        // 3. 快指针找到有效元素时，慢指针移动并更新
        if (nums[fast] != nums[slow]) {
            slow++;
            nums[slow] = nums[fast]; // 覆盖重复值
        }
        // 4. 快指针继续遍历（无需操作，for循环自动++）
    }
    
    // 5. 慢指针+1为有效长度
    return slow + 1;
}
```

经典例题：链表找环

```java
// LeetCode 141：判断链表是否有环
class ListNode {
    int val;
    ListNode next;
    ListNode(int x) { val = x; }
}

public boolean hasCycle(ListNode head) {
    if (head == null || head.next == null) return false;
    
    // 慢指针走1步，快指针走2步
    ListNode slow = head;
    ListNode fast = head.next;
    
    while (slow != fast) {
        // 快指针先到末尾，无环
        if (fast == null || fast.next == null) return false;
        slow = slow.next;
        fast = fast.next.next;
    }
    return true; // 相遇则有环
}
```
