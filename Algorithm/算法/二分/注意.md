*   基础二分（找目标值）：闭区间 `[left, right]`，找到即返回，核心是正确收缩左右边界。

*   左边界二分：找到目标后不返回，继续 `right = mid - 1`，记录最后一次匹配的索引。

*   右边界二分：找到目标后不返回，继续 `left = mid + 1`，记录最后一次匹配的索引。

*   mid 的计算：必须用 `left + (right - left) / 2`，而不是 `(left + right) / 2` —— 避免 `left + right` 超出 int 范围（比如 left 和 right 都是 1e9 时）。

*   区间收缩：

    *   闭区间 `[left, right]`：循环条件是 `left <= right`，更新时 `left = mid + 1` / `right = mid - 1`。
    *   左闭右开 `[left, right)`：循环条件是 `left < right`，更新时 `left = mid + 1` / `right = mid`（面试中闭区间更直观，推荐用）。

*   边界查找：找到目标值后不终止，而是继续收缩区间（左边界往左、右边界往右）。
