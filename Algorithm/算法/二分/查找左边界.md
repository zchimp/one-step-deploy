有序数组中有重复元素，找目标值第一次出现的位置（比如 \[1,2,2,2,3] 找 2 的左边界）

```java
public int findLeftBound(int[] nums, int target) {
    int left = 0;
    int right = nums.length - 1;
    // 记录结果，初始为-1（未找到）
    int res = -1;

    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] == target) {
            // 找到目标，不直接返回，继续向左找更小的索引
            res = mid;
            right = mid - 1;
        } else if (nums[mid] < target) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }
    return res;
}
```
