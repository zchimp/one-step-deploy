有序数组中有重复元素，找目标值最后一次出现的位置

```java
public int findRightBound(int[] nums, int target) {
    int left = 0;
    int right = nums.length - 1;
    int res = -1;

    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] == target) {
            // 找到目标，不直接返回，继续向右找更大的索引
            res = mid;
            left = mid + 1;
        } else if (nums[mid] < target) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }
    return res;
}
```
