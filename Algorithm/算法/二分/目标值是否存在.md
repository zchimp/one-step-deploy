有序数组中查找某个具体值，找到返回索引，找不到返回 -1。

1.  初始化左右指针，覆盖整个数组
2.  循环条件：区间不为空（left <= right）
3.  找到目标，直接返回索引
4.  目标在右半区，收缩左边界（mid已验证不是目标，所以left=mid+1）
5.  目标在左半区，收缩右边界（同理right=mid-1）
6.  循环结束未找到，返回-1

```java
public class BinarySearch {
    // 闭区间 [left, right] 版本（最常用）
    public int binarySearch(int[] nums, int target) {
        // 1. 初始化左右指针，覆盖整个数组
        int left = 0;
        int right = nums.length - 1;
        
        // 2. 循环条件：区间不为空（left <= right）
        while (left <= right) {
            // 计算mid：避免 (left + right) 溢出，等价于 (left + right) / 2
            int mid = left + (right - left) / 2;
            
            if (nums[mid] == target) {
                // 3. 找到目标，直接返回索引
                return mid;
            } else if (nums[mid] < target) {
                // 4. 目标在右半区，收缩左边界（mid已验证不是目标，所以left=mid+1）
                left = mid + 1;
            } else {
                // 5. 目标在左半区，收缩右边界（同理right=mid-1）
                right = mid - 1;
            }
        }
        // 6. 循环结束未找到，返回-1
        return -1;
    }

    // 测试用例
    public static void main(String[] args) {
        BinarySearch bs = new BinarySearch();
        int[] nums = {1, 3, 5, 7, 9, 11};
        System.out.println(bs.binarySearch(nums, 5));  // 输出：2
        System.out.println(bs.binarySearch(nums, 6));  // 输出：-1
    }
}
```
