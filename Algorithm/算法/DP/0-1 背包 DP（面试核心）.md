适用场景
每个物品选 / 不选，求最大价值 / 恰好装满的方案数（比如分割等和子集、目标和）。

```java
// 示例：0-1背包（求最大价值）
public int knapsack01(int[] weights, int[] values, int capacity) {
    int n = weights.length;
    // 1. 状态定义：dp[j]表示容量为j的背包能装的最大价值（空间优化：一维数组）
    int[] dp = new int[capacity + 1];
    
    // 2. 初始化：dp[0] = 0，其余默认0
    
    // 3. 遍历顺序：物品正序，容量倒序（避免重复选同一物品）
    for (int i = 0; i < n; i++) { // 遍历物品
        for (int j = capacity; j >= weights[i]; j--) { // 遍历容量（倒序）
            // 4. 状态转移：选或不选当前物品
            dp[j] = Math.max(dp[j], dp[j - weights[i]] + values[i]);
        }
    }
    
    // 5. 返回结果
    return dp[capacity];
}
```
