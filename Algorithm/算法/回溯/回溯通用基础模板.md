这是所有回溯题的「母模板」，只需调整「选择列表」「终止条件」「剪枝条件」即可适配所有场景：

```java
/**
 * 回溯通用模板
 * @param nums/str  输入的原始数据（数组/字符串）
 * @param path      当前路径（记录已选元素）
 * @param result    最终结果集（存储所有合法路径）
 * @param start     起始索引（组合/子集用，排列不用）
 * @param used      已使用标记（排列去重用）
 */
public void backtrack(
    输入数据类型 input, 
    路径容器 path, 
    结果容器 result, 
    int start, 
    boolean[] used
) {
    // 1. 终止条件：路径满足要求，加入结果集（核心出口）
    if (终止条件) { // 比如：path.size() == k 或 遍历完所有元素
        result.add(new ArrayList<>(path)); // 必须拷贝，避免引用覆盖
        return;
    }

    // 2. 遍历选择列表（核心：尝试所有可能的选择）
    for (int i = start; i < input.length; i++) { // 排列用i=0，组合/子集用i=start
        // 2.1 剪枝条件（可选但关键！减少无效递归）
        if (剪枝条件) { // 比如：元素重复、路径和超过target、已使用过
            continue;
        }

        // 2.2 做出选择：将当前元素加入路径，标记已使用
        path.add(input[i]);
        if (used != null) used[i] = true;

        // 2.3 递归探索下一层：注意start的传递（组合/子集i+1，排列0）
        backtrack(input, path, result, i + 1, used); // 排列传0，组合/子集传i+1

        // 2.4 撤销选择（回溯核心！恢复状态，尝试下一个选择）
        path.remove(path.size() - 1);
        if (used != null) used[i] = false;
    }
}
```

模板使用说明（关键参数）
表格
参数	排列问题	组合问题	子集问题
start	0	i+1	i+1
used	必须用	不用	不用
剪枝	去重	去重 / 限长	去重
