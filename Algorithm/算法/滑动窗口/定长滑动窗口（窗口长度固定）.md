找长度为 k 的子数组 / 子串的最大和、最小和、包含特定字符的数量等（LeetCode 643、1456）。

```java
// 示例：LeetCode 643 - 子数组最大平均数 I（定长k）
public double findMaxAverage(int[] nums, int k) {
    int left = 0, right = 0;
    int windowSum = 0;
    double maxAvg = Double.MIN_VALUE;
    
    // 先扩张窗口到长度k
    while (right < nums.length) {
        windowSum += nums[right];
        right++;
        
        // 窗口长度达到k时，计算结果并收缩左指针
        if (right - left == k) {
            maxAvg = Math.max(maxAvg, (double) windowSum / k);
            windowSum -= nums[left];
            left++;
        }
    }
    return maxAvg;
}
```
