# 刷入recovery
需要解除设备锁，下载android sdk套件，支持adb fastboot等命令

下载twrp软件
https://twrp.me/Devices/ 选择机型
redmi note 7 pro（TWRP for violet）: https://twrp.me/xiaomi/xiaomiredminote7pro.html

xiaomi mi 8（TWRP for dipper）：https://twrp.me/xiaomi/xiaomimi8.html
执行
adb reboot bootloader
fastboot flash recovery twrp.img
fastboot reboot



# 面具获取root权限
步骤：

一、小米解锁 Bootloader：Xiaomi-unlock
如果你的手机不能解锁BL，推荐 光速虚拟机（不用解锁BL也可以刷面具）

二、下载系统包：rom（一定要下载和手机系统版本一致的系统包）

三、提取 boot.img ：payload-dumper-go-boot（如果系统包有 boot.img，可以跳过此步骤）

四、手机插电脑，文件传输模式，复制 boot.img 到手机 Download 目录

五、手机下载 Magisk APP 安装：Magisk-download

六、打开 Magisk，1：安装 – 2：选择 boot.img – 3：开始修补文件 – 4：修补成功
